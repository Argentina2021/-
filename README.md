# 百家数科 — 气泡交互展示

雄安科技展企业气泡交互页面，1920×1080 全屏展示。支持**一体机 + 大屏**双屏联动模式。

## 功能

- Canvas 渲染 3D 半透明气泡，企业名称显示在气泡内（MiSans VF）
- 空中漂浮物理：浮力、重力、空气阻力、气流扰动
- 空间纵深：前后虚实、视差
- 背景视频 + plus-lighter 混合
- **双屏联动**：一体机滑动/点击，大屏实时同步
- 点击一体机气泡 → 气泡变大变亮，大屏展开企业产品卡片

## 双屏部署

在同一台电脑上打开两个浏览器窗口（或分别连接一体机触摸屏与大屏显示器）：

| 界面 | 地址 | 用途 |
|------|------|------|
| 一体机操控 | http://localhost:8080/bubbles-control.html | 触摸滑动浏览、点击选中气泡 |
| 大屏展示 | http://localhost:8080/bubbles-display.html | 同步气泡滚动、展示企业详情卡片 |

右上角连接状态指示：绿色为双屏已联通。

### 交互说明

- **滑动**：在一体机界面左右滑动，两个界面的气泡同步滚动
- **选中**：点击一体机上的气泡，该气泡放大并高亮，其余气泡变暗；大屏自动弹出对应企业信息
- **取消选中**：再次点击同一气泡，或点击空白区域
- **卡片导航**：在大屏上可用左右箭头切换企业（一体机选中状态同步更新）

## 本地预览

```bash
python -m http.server 8080
```

| `bg.mp4` | 背景视频 |
